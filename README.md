# customer-churn-prediction using Machine Learning

## 📌 Problem Statement

Customer churn is a major issue for businesses, especially in the telecom industry, where customers stop using a company's services. The goal of this project is to predict whether a customer will churn or not based on various features such as demographics, usage patterns, and account information.


---


## ⚙️ Methodology

1. Data Collection and Loading

2. Data Cleaning and Preprocessing

3. Exploratory Data Analysis (EDA)

4. Feature Engineering

5. Model Building (Logistic Regression, Decision Tree, Random Forest)

6. Model Evaluation using accuracy, confusion matrix, and classification report


---


## 🚀 Technologies Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Scikit-learn


---


## 🔑 Key Features

Predicts whether a customer will churn or not

Uses Random Forest machine learning model for prediction

Includes data visualization for better understanding

Provides model evaluation metrics for performance analysis

Identifies important factors affecting churn(feature importance)


---


## 📊 Results

Achieved good accuracy using Random Forest model

Identified important features affecting customer churn
